QUD Referral Program

Регламент использования QUD Referral Code и предоставления реферальных вознаграждений

Настоящий Регламент определяет условия участия трейдеров в закрытой реферальной программе QUD, порядок использования Strategy ID и Referral Code, а также правила предоставления вознаграждений и бонусов.

Редакция v0.3 QUD Referral

Ключевые условия программы

Участие

Referral Code предоставляется только владельцу стратегии с присвоенным Strategy ID.

Активность

Для действия кода необходимо сохранять подписку, торговую активность и статус QUD Room или QUD Elite.

Вознаграждение

За одного нового уникального трейдера предоставляется только одно реферальное вознаграждение.

QUD Room

Один подтверждённый новый участник — один бесплатный месяц подписки QUD.

QUD Elite

Вознаграждение определяется в виде процента от первой подтверждённой покупки TDA Basic.

Приглашённый трейдер

После первой отчётной недели получает возможность сопоставить свои показатели со средними значениями трейдеров QUD.

Раздел 01

Общие положения

QUD Referral Program — закрытая реферальная программа для трейдеров, которые сформировали подтверждённую накопительную торговую историю в QUD.

Программа позволяет владельцу торговой стратегии приглашать новых трейдеров к подключению TDA Basic и получать вознаграждение за каждого нового приглашённого трейдера, выполнившего условия настоящего Регламента.

Основной принцип Участие в реферальной программе не предоставляется автоматически всем пользователям QUD.
Раздел 02

Условия участия в реферальной программе

Участвовать в QUD Referral Program может только трейдер, торговой стратегии которого присвоен персональный Strategy ID.

Первичное присвоение Strategy ID возможно только при одновременном выполнении следующих условий:

  • трейдер непрерывно торгует и формирует накопительную историю в QUD не менее двух месяцев;
  • торговая стратегия находится в QUD Room;
  • QUD располагает достаточным объёмом торговых данных для подтверждения непрерывности накопительной истории;
  • у трейдера отсутствуют нарушения, препятствующие участию в программе.

После перехода стратегии из QUD Room в QUD Elite право на участие в реферальной программе сохраняется.

Раздел 03

Strategy ID

Strategy ID — постоянный идентификатор торговой стратегии внутри QUD.

Strategy ID:

  • связывает торговую стратегию с её накопительной историей;
  • используется для внутренней идентификации данных, отчётов и статуса стратегии;
  • является конфиденциальным;
  • не должен публиковаться или передаваться третьим лицам.
Обязательное условие Наличие Strategy ID является обязательным условием получения Referral Code.
Раздел 04

Referral Code

Referral Code — персональный код владельца торговой стратегии, предназначенный для приглашения новых трейдеров в TDA Basic.

Referral Code:

  • связан с конкретным Strategy ID;
  • принадлежит конкретному владельцу;
  • может передаваться потенциальным участникам TDA Basic;
  • может использоваться многократно для приглашения разных трейдеров;
  • не может быть передан другому владельцу.

Один Referral Code может использоваться для приглашения неограниченного количества разных трейдеров при соблюдении условий настоящего Регламента.

Раздел 05

Условия действия Referral Code

Для продолжения участия в реферальной программе владелец Referral Code должен одновременно соблюдать следующие условия:

  • иметь действующую оплаченную подписку на TDA Basic либо на другой платный отчётный продукт QUD, включённый в реферальную программу;
  • продолжать торговую деятельность;
  • формировать новые торговые данные и поддерживать актуальность накопительной истории;
  • находиться в QUD Room или QUD Elite;
  • соблюдать настоящий Регламент.
Ограничение Сам факт наличия Strategy ID или ранее полученного Referral Code не предоставляет бессрочного права на участие в программе.

Действие Referral Code может быть временно приостановлено, если его владелец:

  • прекратил оплачивать подписку на отчётный продукт QUD;
  • прекратил торговую деятельность;
  • в течение продолжительного периода не формирует новые торговые данные;
  • покинул QUD Room или QUD Elite;
  • нарушил условия настоящего Регламента.

Периоды отсутствия сделок, обусловленные особенностями торговой стратегии, могут рассматриваться QUD индивидуально.

После восстановления оплаченной подписки, торговой активности и допустимого уровня участия действие Referral Code может быть восстановлено.

Ранее подтверждённые вознаграждения сохраняются. Новые подключения, совершённые в период приостановки Referral Code, могут не учитываться.

Раздел 06

Порядок использования Referral Code

Приглашённый трейдер должен указать Referral Code при первом подключении и первой оплате TDA Basic.

Важно Referral Code нельзя добавить к покупке после подтверждения оплаты.

Один приглашённый трейдер учитывается в реферальной программе только один раз.

Повторные оплаты, продления подписки или приобретение других продуктов тем же приглашённым трейдером не создают нового вознаграждения, если QUD отдельно не установил иные условия.

Не допускаются:

  • использование владельцем собственного Referral Code;
  • создание повторных аккаунтов одним человеком;
  • использование нескольких email для повторного получения вознаграждений или бонусов;
  • закрепление одного приглашённого трейдера одновременно за несколькими Referral Code;
  • использование чужих персональных данных;
  • фиктивные покупки или подключения.

QUD вправе проверять уникальность приглашённого трейдера по доступным идентификационным и торговым данным.

Раздел 07

Условия вознаграждения владельца Referral Code

Право на вознаграждение возникает после первой подтверждённой оплаты TDA Basic новым уникальным приглашённым трейдером.

Для получения вознаграждения должны одновременно выполняться следующие условия:

  • Referral Code действовал на момент подключения приглашённого трейдера;
  • владелец Referral Code имел действующую оплаченную подписку QUD;
  • владелец продолжал торговую деятельность;
  • приглашённый трейдер впервые подключился к TDA Basic;
  • приглашённый трейдер указал Referral Code до подтверждения оплаты;
  • оплата TDA Basic была получена и подтверждена;
  • приглашённый трейдер ранее не учитывался в реферальной программе;
  • отсутствуют признаки дублирования, фиктивной регистрации или другого нарушения.
Правило начисления За одного уникального приглашённого трейдера владелец получает только одно вознаграждение.

Тип вознаграждения определяется уровнем владельца Referral Code на дату подтверждения первой покупки приглашённого трейдера.

Раздел 08

Вознаграждение владельца в QUD Room

Владелец Referral Code, находящийся в QUD Room, получает за каждого нового приглашённого трейдера:

Вознаграждение QUD Room Бесплатное продление своей подписки QUD на один месяц.

Один новый приглашённый трейдер, впервые оплативший TDA Basic, предоставляет владельцу один дополнительный бесплатный месяц подписки.

Бесплатные месяцы могут накапливаться без ограничения по общему количеству, однако использовать их можно не более трёх месяцев подряд.

После использования трёх бесплатных месяцев подряд владелец должен самостоятельно оплатить не менее одного полного месяца подписки.

После самостоятельно оплаченного месяца владелец может использовать следующий блок накопленных бесплатных месяцев — снова не более трёх месяцев подряд.

Неиспользованные бесплатные месяцы:

  • не сгорают;
  • сохраняются на балансе владельца;
  • переносятся на следующий доступный период;
  • используются после выполнения условия об одном самостоятельно оплаченном месяце между бесплатными блоками.
Пример
  • три приглашённых трейдера — три бесплатных месяца подряд;
  • после них владелец самостоятельно оплачивает один месяц;
  • затем может использовать следующие накопленные бесплатные месяцы — снова не более трёх подряд.

Если накоплено более трёх бесплатных месяцев, оставшиеся месяцы сохраняются до следующего доступного периода.

Бесплатное продление:

  • применяется после завершения текущего оплаченного периода;
  • не выплачивается денежными средствами;
  • не обменивается на денежный эквивалент;
  • не передаётся другому лицу;
  • не может быть продано или подарено.
Раздел 09

Вознаграждение владельца в QUD Elite

Владелец Referral Code, находящийся в QUD Elite, получает за каждого нового приглашённого трейдера:

Вознаграждение QUD Elite Денежное вознаграждение в виде процента от суммы его первой подтверждённой покупки TDA Basic.

Размер процента устанавливается действующими условиями QUD Elite и сообщается владельцу отдельно.

Процентное вознаграждение рассчитывается только от первой подтверждённой покупки TDA Basic приглашённого трейдера.

Повторные оплаты и продления того же приглашённого трейдера не создают повторной процентной выплаты, если QUD отдельно не утвердил другие условия.

Порядок, способ и срок выплаты сообщаются владельцу Referral Code в рамках условий QUD Elite.

Раздел 10

Бонус приглашённому трейдеру

Трейдер, который использовал действующий Referral Code при первом подключении и первой оплате TDA Basic, получает специальный бонус.

Бонус приглашённому трейдеру После первой отчётной недели трейдер сможет сравнить свои показатели со средними значениями трейдеров QUD.

Приглашённому трейдеру предоставляются:

  • его показатели за первую отчётную неделю;
  • соответствующие средние значения трейдеров QUD.

QUD предоставляет только фактические значения показателей.

QUD:

  • не оценивает различия между показателями;
  • не определяет, какие значения являются правильными или неправильными;
  • не формирует рейтинг приглашённого трейдера;
  • не делает выводов о качестве его торговой стратегии;
  • не предоставляет торговых рекомендаций;
  • не указывает, каким образом трейдер должен изменить свою торговлю.

Все выводы и интерпретации трейдер делает самостоятельно.

Бонус:

  • предоставляется один раз;
  • предоставляется после первой полноценной отчётной недели;
  • доступен только при использовании Referral Code во время первого подключения;
  • не предоставляется при добавлении Referral Code после оплаты;
  • не имеет денежного эквивалента;
  • не заменяется другим продуктом или услугой.
Раздел 11

Подтверждение вознаграждений и бонусов

QUD самостоятельно фиксирует:

  • факт использования Referral Code;
  • подтверждение первой оплаты TDA Basic;
  • уникальность приглашённого трейдера;
  • право владельца на вознаграждение;
  • тип вознаграждения владельца;
  • количество накопленных бесплатных месяцев;
  • количество использованных подряд бесплатных месяцев;
  • наличие обязательного самостоятельно оплаченного месяца между бесплатными блоками;
  • готовность сравнительных показателей приглашённого трейдера;
  • факт предоставления вознаграждения;
  • факт направления сравнительных показателей.
Подтверждение QUD Вознаграждение владельца или бонус приглашённому трейдеру считаются предоставленными только после их подтверждения со стороны QUD.
Раздел 12

Возврат оплаты и отмена вознаграждения

Если оплата приглашённого трейдера была возвращена, отменена или признана недействительной, QUD вправе отменить связанное с ней вознаграждение.

Если бесплатный месяц уже был начислен, QUD вправе скорректировать количество накопленных месяцев или срок последующего продления.

Если денежное вознаграждение ещё не было выплачено, оно может быть отменено.

Если нарушение выявлено после выплаты, QUD вправе учесть соответствующую сумму при последующих выплатах.

Раздел 13

Нарушения условий программы

Нарушением считается:

  • самостоятельное использование владельцем собственного Referral Code;
  • создание фиктивных или повторных аккаунтов;
  • использование нескольких email одним человеком;
  • предоставление недостоверной информации;
  • попытка получить несколько вознаграждений за одного трейдера;
  • имитация торговой активности исключительно для сохранения участия в программе;
  • передача Strategy ID третьим лицам;
  • продажа или передача права на вознаграждение;
  • иное злоупотребление условиями программы.

При выявлении нарушения QUD вправе:

  • временно приостановить действие Referral Code;
  • полностью прекратить участие владельца в программе;
  • отменить неподтверждённое вознаграждение;
  • исключить отдельное подключение из реферальной программы;
  • отказать в повторном восстановлении действия Referral Code.
Раздел 14

Изменение условий программы

QUD вправе изменять:

  • условия участия в реферальной программе;
  • требования к торговой активности;
  • перечень продуктов, включённых в программу;
  • форму и размер вознаграждения;
  • порядок использования накопленных бесплатных месяцев;
  • процентное вознаграждение для QUD Elite;
  • содержание бонуса приглашённому трейдеру;
  • порядок подтверждения и предоставления вознаграждений.

Новые условия применяются к подключениям, совершённым после вступления изменений в силу.

Ранее подтверждённые вознаграждения сохраняются, если их отмена не связана с возвратом оплаты, дублированием аккаунтов или нарушением настоящего Регламента.

Раздел 15

Принятие условий

Получение и использование Referral Code означает, что его владелец:

  • ознакомился с настоящим Регламентом;
  • понимает условия участия;
  • принимает правила предоставления вознаграждений;
  • обязуется поддерживать действующую подписку и торговую активность;
  • соглашается с правом QUD проверять соблюдение условий программы.
Принятие Регламента Передача Referral Code приглашённому трейдеру допускается только в рамках настоящего Регламента.
