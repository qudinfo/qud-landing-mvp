# QUD Landing

Первая техническая версия лендинга QUD Trade Decision Audit.

## Структура

- `index.html` — русская версия лендинга
- `style.css` — базовая матовая дизайн-система
- `script.js` — заготовка под будущие функции
- `assets/` — будущие изображения, иконки и логотипы

## Разделы лендинга

- Аудит
- Трейдерам
- Инвесторам
- О QUD
- Контакты
